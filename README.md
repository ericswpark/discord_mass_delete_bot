# Discord Mass Delete Bot

Bot to delete messages in given channel

## Usage

 - Obtain a bot API key
 - Put it in `auth.json` file in the following format:

```
{
    "token": "XXXXX-token-here-XXXXXX"
}
```

 - Start the app using

```
node bot.js
```
